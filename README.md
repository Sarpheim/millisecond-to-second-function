# millisecond-to-second-function
Millisecond to second function.

---

## Usage - Exapmle 1
```javascript
const MillisecondToSecondFunction = require("./millisecond-to-second-function.js")
const millisecond = 1250

console.log(MillisecondToSecondFunction(millisecond))
```

---

## Result - Example 1
```
1.250
```
